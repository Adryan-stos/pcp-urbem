# PCP Urbem - Estrutura Inicial

Sistema para Controle de Projetos e Programação e produção da Fábrica 2.


## Estrutura

- `src/pages`: telas principais
- `src/components`: componentes reutilizáveis
- `src/lib`: conexão com Supabase
- `src/services`: funções de consulta/salvamento
- `src/styles`: estilos globais
